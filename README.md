## Description

Application that automates the ordering process of a pizzeria.

## Swagger docs

http://localhost:3999/docs

## Clone repository

```bash
$ git clone https://github.com/maycomorais/mayconburger.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
